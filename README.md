# c-system-programming-chat


This repository contains a local chat application built in C. It uses named pipes (`mkfifo`) for inter-process communication, allowing multiple clients to connect to a central server and exchange messages in real-time.

## Features

* **Multi-Client Support**: The server can handle up to 20 active clients simultaneously[cite: 6]. 
* **Real-Time Broadcasting**: Messages sent by any client are instantly broadcasted to all other connected clients through the server[cite: 6].
* **Server Logging**: The server automatically records all chat activity, joins, and leaves into a log file with exact timestamps[cite: 6].
* **Threaded Client**: The client application uses `pthread` to read incoming messages while simultaneously waiting for user input.

## Project Structure

* `server.c`: The main server application that creates the central `server_fifo`, manages active clients, and broadcasts messages[cite: 6].
* `client.c`: The client application that creates a unique client FIFO (using its Process ID) and connects to the server[cite: 3].
* `chat_log.txt`: The output log file generated by the server to track chat history[cite: 1, 6].

## Prerequisites

To compile and run this project, you need:
* A C compiler (like GCC).
* A POSIX-compliant operating system (Linux/macOS) since it relies on `sys/stat.h`, `pthread`, and FIFOs[cite: 3, 6].
